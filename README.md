# 2020_Makerthon
2020_makerthon에 출전한 개발 내용을 기록합니다.

-----------------------------------------------

출전 제품인 **Pillit**의 소개가 **pillit_introduction.pdf**에 간략히 정리되어있습니다.

개발 내용은 아래와 같습니다.

## pillit system
- pilliter
  - 스마트 디스펜서
  - Arduino를 이용하여 개발
- pillitee
  - 스마트 약통
  - Arduino를 이용하여 개발
- App
  - pilliter와 pillitee 제어
  - AI 알람 서비스 제공 (AI Model이 사용자의 복용 예상 시간을 학습하여 알람을 자동 설정)
  - MIT AppInventor로 제작
  - Google SpreadSheet를 이용하여 AI model과 사용자의 복용정보 공유
- AI
  - user의 복용주기 학습
  - 사용자의 복용 예상 시간을 예측하여 제공
  - pycharm을 통해 코드작성
  - Google SpreadSheet와 연동하여 앱과 사용자의 복용 정보 공유
